Calculator

Building a calculator was my first ever project using Javascript. With my calculator you can press buttons on the screen to do calculations. It only works with two numbers and it can’t be typed

This was a challenging project since I was still very inexperienced in Javascript. I had to learn “onclick” and submit events for the code to work. Also lots of brainstorming to uniquely craft my calculator with the code in my arsenal. After finishing the interactive part, I just need to do simple CSS to make it look like a calculator.
