Media Query

Media query is a skill that I would say that all programmers should all have. It basically allows our website to work on mobile. How this works is it will detect how big the window size is. Depending on the size, style on the content will change. For example, if three images are next to each other on the desktop, on mobile we can change it to become an image carousel so all three images can be displayed on a smaller screen.
